# SingleSpa 批量管理

- 项目名称: single-spa-template
- 说明: 使用 npm-run-all 工具，批量安装 node modules，批量启动主应用和子应用

## 克隆代码

本仓库包含所有主应用和子应用

```shell
git clone https://github.com/gokingliu/single-spa-template.git
```

## 安装 npm-run-all

```shell
npm i
```

## 批量管理主应用和子应用

- 批量安装

```shell
npm run install-all
```

- 批量 vite 运行

```shell
npm run start-vite
```

- 批量 webpack 运行

```shell
npm run start-webpack
```

## 启动后本地预览地址

http://localhost:8880
